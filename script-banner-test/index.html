Script Banner Test

En esta aplicación web podrás verificar tus banners con tu código de javascript proporcionado por tu servidor de banners.

Ten en cuenta que esta aplicación web sirve únicamente para corroborar que el banner este animado correctamente y que se muestre adecuadamente según los estándares de la IAB México y DoubleClick de Google.

Si el banner no se muestra debes de verificar que el código es correcto y que la campaña siga activa.

Esta herramienta se debe de usar adecuadamente solo para probar que el banner se muestra correctamente y no se debe de usar la aplicación con fines de aumentar el número de impresiones y/o número de clicks del banner.

Se debe de tener en cuenta que un banner en estructura o código <script> tiene una similutud o igualdad la siguiente código:

<!-- Inicio banner formato SCRIPT -->
    <SCRIPT language='JavaScript1.1' SRC="https://ad.doubleclick.net/ddm/adj/N9445.2010504CEREBROSMARTMEDIA/B10446234.140367884;sz=300x600;ord=[timestamp];dc_lat=;dc_rdid=;tag_for_child_directed_treatment=?"></SCRIPT>
    <NOSCRIPT>
        <A HREF="https://ad.doubleclick.net/ddm/jump/N9445.2010504CEREBROSMARTMEDIA/B10446234.140367884;sz=300x600;ord=[timestamp]?">
            <IMG SRC="https://ad.doubleclick.net/ddm/ad/N9445.2010504CEREBROSMARTMEDIA/B10446234.140367884;sz=300x600;ord=[timestamp];dc_lat=;dc_rdid=;tag_for_child_directed_treatment=?" BORDER=0 WIDTH=300 HEIGHT=600 ALT="Advertisement"/>
        </A>
    </NOSCRIPT>
<!-- Fin banner formato SCRIPT -->


En este espacio se muestra el banner agregado en la parte de arriba.

Explicación a nivel programación

<!-- Inicio banner formato SCRIPT -->
 Este código de "Inicio" como el de "Fin" fungen de documentación exclusivamente para aquellos que necesiten manipular y/o agregar el código en una página de Internet, este código sirve exclusivamente para dividir y encontrar con mayor facilidad el código relacionado con la publicidad. 
<SCRIPT language='JavaScript1.1' SRC="https://ad.doubleclick.net/ddm/adj/N9445.2010504CEREBROSMARTMEDIA/B10446234.140367884;sz=300x600;ord=[timestamp];dc_lat=;dc_rdid=;tag_for_child_directed_treatment=?"></SCRIPT>
 Este código es aquél que manda a llamar al banner desde el servidor de banners, este código se ejecuta únicamente si el navegador del usuario soporta <javascript>. El código no podrá ser ejecutado si el navegador del usuario tiene denegado ejecutar códigos de formato <javascript> o si el navegador actualmente es muy anticuado o viejo. 
<NOSCRIPT>
<A HREF="https://ad.doubleclick.net/ddm/jump/N9445.2010504CEREBROSMARTMEDIA/B10446234.140367884;sz=300x600;ord=[timestamp]?">
<IMG SRC="https://ad.doubleclick.net/ddm/ad/N9445.2010504CEREBROSMARTMEDIA/B10446234.140367884;sz=300x600;ord=[timestamp];dc_lat=;dc_rdid=;tag_for_child_directed_treatment=?" BORDER=0 WIDTH=300 HEIGHT=600 ALT="Advertisement"/>
</A>
</NOSCRIPT>
 Este código está creado para aquellos navegadores que no pudieron cargarse con el anterior código debido a los errores mencionados, este código es como un último recurso para que se pueda mostrar el banner en la mayoría de los navegadores que puedan tener los usuarios.
Nota. Hay que tener en cuenta que cada servidor de banners tiene sus propias especificaciones y en algunos casos se ha observado que también pueden mandar el banner en formato <iframe> pero sigue siendo lo mismo, sólo son diferentes formatos para poder mostrar el banner en el sitio web.

SEO Helper

En esta aplicación web se pretende dar una ayuda al desarrollador para poder "literlamente" copiar y pegar ctrl + c & ctrl + v en el código fuente de la página web.

El código proporcionado por la aplicación está basado con las útlimas tendencias de SEO para poder aprovechar al máximo el desempeño de la página web.

El código lo deberás de modificar con respecto a tu página web o landing page, este código únicamente es para dar una estructura inicial para una buena configuración del SEO.

En el código se hace uso de:

  • Structured Data Markup (JSON-LD format)
  • Metatags
  • Open Graph Support
  • Accelerated Mobile Pages (AMP)
  • Robots
  • Canonical url
  • Atom+xml
